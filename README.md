# my_print
my print library

https://github.com/wallying/my_print.git

## support type:
    %%  - single % character
    %c  - single character
    %s  - string of characters
    %d  - signed decimal integer
    %u  - unsigned decimal integer
    %x  - unsigned hexadecimal integer (lowercase)
    %X  - unsigned hexadecimal integer (uppercase)
