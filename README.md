# my_print
my print library

https://github.com/wallying/my_print.git

##lib_print support format:

    %%      - print % character
    %c      - print character
    %s      - print string
    %d      - print signed decimal integer
    %u      - print unsigned decimal integer
    %x      - print unsigned lower hex integer
    %X      - print unsigned upper hex integer
